# Dhikr & Duas App

A comprehensive app for daily dhikr (remembrance), duas (supplications), prayer times, tasbeeh, and general/combined duas, designed to help users maintain their daily worship in an easy and elegant way.  

## Key Features

1. **Duas & Dhikr**  
   - Categorized duas: Istighfar, Rizq (provision), Shifa (healing), etc.  
   - Quick search to find any dua.  
   - Save favorite duas for easy access.  

2. **General/Combined Dua**  
   - A dua that includes multiple purposes: guidance, blessings, health, provision, etc.  
   - Can be added to favorites, copied, or shared.  

3. **Prayer Times**  
   - Displays prayer times based on the user’s location.  
   - Notifications and alerts before prayer times.  

4. **Tasbeeh Counter**  
   - Digital counter to facilitate daily tasbeeh.  
   - Tracks counts for each session.  

5. **Location (Geo)**  
   - Calculates accurate prayer times according to the user’s city.  

## Technologies Used

- **Frontend**: Android (Jetpack Compose, Kotlin)  
- **Backend**: Firebase / Room Database (optional)  
- **Notifications**: For prayer and dhikr reminders  
- **Location API**: To calculate precise prayer times  

## How to Use

1. Open the app and optionally log in.  
2. Browse duas or use filters to find specific ones.  
3. Use the tasbeeh counter or check prayer times according to your location.
